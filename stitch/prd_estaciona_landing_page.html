# Resumen de la página: Estaciona Landing page para validar una app tipo marketplace de estacionamientos. ## 1. Identidad base * **Nombre:** Estaciona * **Propuesta de valor:** Encuentra o arrienda estacionamientos privados de forma simple y segura. ## 2. Paleta de colores (Oscuro profesional) * Primary: #2563EB (azul) * Primary hover: #1D4ED8 * Accent: #22C55E (verde) * Background: #0B1220 * Surface: #111827 * Text primary: #F9FAFB * Text secondary: #9CA3AF * Border: #1F2937 ## 3. Tipografía * Títulos: Inter o Poppins (Bold/Semibold) * Texto: Inter (Regular) ## 4. Estructura 1. Navbar: Logo y botón "Únete a la lista". 2. Hero: "Arrienda o encuentra estacionamiento en segundos". Botones para anfitrión y conductor. Visual de mockup. 3. Problema: Encontrar estacionamiento es difícil vs espacios vacíos. 4. Cómo funciona: Pasos para conductor y anfitrión. 5. Beneficios: Bloques para ambos tipos de usuario. 6. Confianza: Pagos seguros, perfiles verificados. 7. Visual del producto: Mockups de mapa, reserva y publicación. 8. Waitlist: Formulario de captación (Nombre, Email, Rol). 9. CTA final: "Empieza a aprovechar mejor cada espacio disponible". 10. Footer: Contacto, Redes, Legal. ## 5. Estilo visual * Limpio, mucho espacio en blanco/aire, grid ordenado, bordes redondeados (8-16px), sombras suaves.
